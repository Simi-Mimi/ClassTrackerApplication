# 🛰️ מערכת ניהול ואיכון לטיול שנתי – פרויקט סיום (תוכנית "הסי")

## 📝 אודות הפרויקט
המערכת פותחה כחלק ממטלת בית במסגרת תוכנית "הסי".  
מטרתה היא ניהול לוגיסטי ובטיחותי של טיולים בית-ספריים באמצעות איכון בזמן אמת.

המערכת כוללת:
- ממשק לניהול משתמשים (מורות ותלמידות)
- תצוגה גיאוגרפית חיה
- סימולטור המדמה מכשירי קצה השולחים מיקומי GPS

---

## ✨ תכונות מרכזיות

### 👩‍🏫 ניהול משתמשים
רישום מורות ותלמידות ואחסון בבסיס נתונים רלציוני.

### 🗺️ ניטור גיאוגרפי
הצגת כלל המשתתפים על מפה אינטראקטיבית בזמן אמת.

### ⚠️ מנגנון בטיחות (Geofencing)
זיהוי והתרעה אוטומטית כאשר תלמידה מתרחקת מעבר לטווח המותר.

### 📡 סימולטור איכון
רכיב עצמאי המדמה שליחת נתוני מיקום (GPS) לשרת בזמן אמת.

---

## 🛠️ טכנולוגיות ומבנה המערכת

| שכבה | טכנולוגיה |
|------|------------|
| Client Side | React (JSX) + Leaflet (מפה) |
| Server Side | Java Spring Boot (REST API) |
| Database | H2 Database (SQL, בזיכרון) |
| Simulator | Python (שליחת JSON לשרת) |

---

## 💻 דרישות קדם (Prerequisites)

לפני הרצת הפרויקט יש לוודא התקנה של:

- Java JDK 11 ומעלה + IntelliJ IDEA
- Node.js + npm
- Python 3.x
- VS Code (מומלץ)

---

## ⚙️ הוראות הרצה

### 1. צד שרת (Java Spring Boot)

1. פתח את תיקיית ה-Server ב־IntelliJ IDEA  
2. המתן לטעינת Maven Dependencies  
3. הרץ את הפרויקט (Run Application)
