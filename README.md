# מערכת ניהול ואיכון לטיול
## אודות הפרויקט
מטרת המערכת היא ניהול ובטיחות של טיולים בית- ספריים באמצעות איכון בזמן אמת.

המערכת כוללת:
- ממשק לניהול משתמשים (מורות ותלמידות)
- תצוגה גיאוגרפית חיה
- סימולטור המדמה מכשירי איכון תלמידים השולחים מיקומי GPS


## תכונות מרכזיות  

###  ניהול משתמשים
רישום מורות ותלמידות ואחסון בבסיס נתונים.

###  ניטור גיאוגרפי
הצגת כלל המשתתפים על מפה אינטראקטיבית בזמן אמת.

###  מנגנון בטיחות (Geofencing)
זיהוי והתרעה אוטומטית כאשר תלמידה מתרחקת מעבר לטווח המותר.

### סימולטור איכון
רכיב עצמאי המדמה שליחת נתוני מיקום (GPS) לשרת בזמן אמת.

---

## טכנולוגיות ומבנה המערכת

| שכבה | טכנולוגיה |
|------|------------|
| Client Side | React (JSX) + Leaflet (מפה) |
| Server Side | Java Spring Boot (REST API) |
| Database | H2 Database (SQL) |
| Simulator | Python (  לשרת JSON שליחת) |

---

## 💻 דרישות קדם (Prerequisites)

לפני הרצת הפרויקט יש לוודא התקנה של:

- Java JDK 11 ומעלה + IntelliJ IDEA
- Node.js + npm
- Python 3.x
- VS Code (מומלץ)

---

## הוראות הרצה

### 1. צד שרת (Java Spring Boot)

1. פתח את תיקיית ה-Server ב־IntelliJ IDEA  
2. המתן לטעינת Maven Dependencies  
3. הרץ את הפרויקט (Run Application)

גישה למסד הנתונים:
http://localhost:8080/h2-console

יש לוודא התאמה ל־JDBC URL בקובץ `application.properties`.

---

### 2. צד לקוח (React)

1. פתח את תיקיית ה-Client ב־VS Code  
2. פתח טרמינל והריץ:


npm install

הפעל את הפרויקט:
npm run dev
פתח את הקישור שיופיע (בדרך כלל):
http://localhost:5173
סימולטור (Python)
פתח את תיקיית הסימולטור ב־VS Code
הרץ:
python simulator.py

מבנה ה-API

המערכת חושפת את ה־Endpoints הבאים:

GET /api/students – שליפת רשימת תלמידות (עבור מורות)
POST /api/location – קבלת עדכוני מיקום מהסימולטור
POST /api/register – רישום משתמשים חדשים

הערה

הפרויקט הוגש כחלק ממטלת בית במסגרת תוכנית "הסי".
