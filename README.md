#  Class Tracker Application
##טכנולוגיות
* **Client:** React
* **Server:** Java Spring Boot
* **Database:** SQL H2
* **Simulator:** Python
## 🛠 טכנולוגיות בשימוש
* **צד לקוח:** React.js (שימוש ב-Leaflet/Google Maps עבור המפה).
* **צד שרת:** Java Spring Boot.
* **מאגר מידע:** SQL H2 (In-Memory).
* **סימולטור:** Python (שולח נתוני מיקום בפורמט JSON).
